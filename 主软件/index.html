05 售后维修安装

买得放心，送装维修都有人管

DiliLife 帝力生活把商品售后、安装预约、维修报价和师傅评价并入主平台。用户提交需求后，客服通过 WhatsApp 确认服务内容、上门时间和预计费用。

WhatsApp 预约

服务闭环

用户预约 -> 客服确认 -> 报价确认 -> 派单师傅 -> 上门服务 -> 完工验收 -> 质保复查

  1. 用户上传商品、故障或安装位置照片。
  2. 客服确认地址、区域、时间和预计费用。
  3. 师傅到场后，追加配件或施工必须再次确认。
  4. 完工后记录照片、费用、质保日期和用户评价。

用户端

  • 预约安装、申请售后、联系客服。
  • 上门费、人工费、配件费先确认。
  • 质保以服务工单记录为准。

客服后台

  • 从订单或 WhatsApp 创建售后工单。
  • 记录报价、派单、配件、减免和责任方。
  • 投诉、返修和赔付必须闭环。

试运营上线包

  • 上线检查表、样板工单验证表。
  • 真实图片拍摄清单、用户授权确认文案。
  • 商城、库存、配送、财务跨板块同步清单。
